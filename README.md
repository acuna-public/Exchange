# Exchange
 
Exchange is a powerful crypto exchanges API wrapper

Supported exchanges:

- [x] Binance
- [x] Bybit

## Capabilities

- Get prices in every interval
- Get positions info
- Open/Close positions
- TP/SL set & edit
- Switch margin type (cross/isolated)
- Calculate position PNL & ROE
- Calculate liquidation price
- Calculate fees
